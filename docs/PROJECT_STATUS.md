## 教学策略

- 利用 Java 基础对照学习 C#，不重复从变量和循环开始
- 每个工业模块先用模拟器理解协议和异常，再抽象真实设备接口
- 关键代码要求能够独立解释，不把“复制后能运行”作为完成标准
- 项目描述只陈述实际完成和验证的内容，不虚构真实产线或硬件经验
- 默认按零基础讲解，不采用先提问、考试或要求独立作答的方式
- 由指导者分小步实现并验证，再解释代码用途、运行现象和面试表达
- 以尽快具备初级岗位项目能力为优先，暂缓与求职无关的理论扩展
